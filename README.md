# Web-Based-Uno-Python-Challenge-Game
It is a web-based game which includes asking questions on the python language. It acts as an educational game through the rules of Uno. The Uno cards used have QR codes printed to them. The system's tech stack includes Node.js, Express.js, Pug.js and APIs for QR code handling.


- Create and join game sessions.
- QR code cards trigger Python questions.
- Scoreboard to track players' points.
- Host can start and end games.
- Rewards and penalties based on correct answers.
- Fully responsive and interactive front-end with jQuery and HTML5 QR code scanning.
